# 🎮 LERN-DOJO APP — Vollständiges Brief für Claude Code

## Kontext
Zielgruppe: 10–12-jähriger Junge (Anfänger in Field Hockey, Englisch A1-A2)
Plattform: Web-Browser (PC + Tablet, Touch-fähig)
Status: Vollständig online, kein Eltern-Bereich nötig

---

## 🧠 Das Kern-Konzept (NEU — ÜBERARBEITET)

**Name der App: "SHIN DOJO"**
(Shin = japanisch für "wahres/neues Selbst" + Dojo = Trainingsort)

Das ist KEIN normales Lern-Tool. Die App ist ein **Anime-RPG-Universum**.
Der Junge ist ein Lehrling, der zum Meister wird — indem er echte Dinge lernt.

### Die große Idee: Wissen = Kraft
Jedes Fach ist eine **Kampfkunst/Disziplin** in der Welt des Shin Dojo:
- 🇬🇧 Englisch = "Die Sprache der Alten" (Kommunikations-Magie)
- ➕ Mathe = "Logik-Jutsu" (Zahlen als Waffe)
- 🏑 Hockey = "Der Weg des Schlägers" (Körper-Disziplin)
- 🧠 Auswendig = "Gedächtnis-Palast" (Geist-Training)
- ✍️ Schreibschrift = "Kalligraphie-Kunst" (Präzisions-Training)
- 🤖 KI = "Das Geheimnis der Maschinen" (Weisheits-Pfad)
- ❓ Quiz = "Der Arena-Kampf" (Test aller Fähigkeiten)

## 🆕 NEUE IDEEN (noch nicht besprochen — vom Planer)

### 1. Rang-System statt simplen "Levels"
Nicht "Level 5" sondern echte Ninja-Ränge:
- 🥋 Schüler (Lvl 1-5)
- ⚔️ Krieger (Lvl 6-15)
- 🌟 Meister (Lvl 16-30)
- 👑 Legende (Lvl 31-50)
- 🔱 Dojo-Meister (Lvl 51+)

Jeder Rangaufstieg = große Animation + seltenes Avatar-Item

### 2. Combo-Multiplikator-System
Wie in einem Beat-em-up:
- 3 richtige Antworten hintereinander → 2× Münzen
- 5 richtig → 3× Münzen + "Combo-Feuer" Animation
- 10 richtig → 5× Münzen + "PERFEKT!" Spezialeffekt
- Falsche Antwort = Combo bricht

### 3. Missions-Ranking (Anime-Style)
Statt "Tagesauftrag" — echte **Rang-Missionen**:
- 📜 C-Rang: 3 Vokabeln lernen (Bronze-Belohnung)
- 📜 B-Rang: Mathe-Quiz bestehen (Silber-Belohnung)
- 📜 A-Rang: Alle Tages-Module erledigen (Gold-Belohnung)
- 📜 S-Rang: Wöchentlicher Boss-Kampf (Seltenes Item)

### 4. Wöchentlicher Boss-Kampf
Jeden Sonntag erscheint ein "Wissens-Boss" (animierte Figur).
Er greift an mit Fragen aus ALLEN Fächern.
Der Spieler antwortet → macht Schaden → besiegt den Boss.
Belohnung: Exklusives Avatar-Outfit das nur durch Boss-Kampf erhältlich ist.

### 5. Wissens-Weltkarte
Auf einer animierten Karte gibt es verschiedene "Reiche":
- Jedes Reich = ein Fach
- Wenn er Lektionen abschließt, "erkundet" er das Reich
- Neue Gebiete öffnen sich mit höherem Level
- Versteckte Geheimnisse und Schätze in der Karte

### 6. Gedächtnis-Palast (für Auswendig-Lernen)
Visuelles Lern-System: Er platziert Dinge in virtuellen Räumen.
- Zimmer 1: Vokabeln (hängen an Wänden)
- Zimmer 2: Formeln (stehen auf Tafeln)
- Zimmer 3: Texte (liegen auf Tischen)
Er "geht" durch den Palast um abzurufen was er gelernt hat.
Wissenschaftlich belegt als effektivste Merkstrategie.

### 7. KI-Sensei (lebendiger Charakter)
Kein langweiliges "Was ist KI"-Tutorial.
Stattdessen: Ein **animierter Sensei-Charakter** der durch die App führt.
- Er heißt "Shin-Sensei" und sieht aus wie ein weiser alter Anime-Meister
- Er erklärt Konzepte mit Witz und einfachen Worten
- Er IST die KI — also lernt der Junge KI indem er mit ihr interagiert
- Bei schwierigen Aufgaben sagt er: "Lass mich dir helfen..."
- Er gibt Hinweise, lobt, korrigiert — aber niemals langweilig

### 8. Hockey-Taktik-Board (interaktiv)
Kein Text über Taktiken — ein echter interaktiver Spielfeldeditor:
- Er kann Spielerpositionen auf einem Feld platzieren
- Bewegungspfeile zeichnen
- Verschiedene Standardformationen lernen (3-3-1, etc.)
- Quiz: "Wo steht der Mittelfeldspieler?" → anklicken auf Feld

### 9. Lernzeit-Intelligenz
Die App misst wann er am besten performed:
- Trackt: Uhrzeit + Ergebnis jeder Session
- Zeigt nach 1 Woche: "Du lernst am besten um 16:00 Uhr!"
- Optimaler Lernzeit-Vorschlag jeden Tag

### 10. Freundschafts-Duell (Später-Feature, jetzt vorbereiten)
Er kann einen Freund herausfordern:
- Teilt einen Code → Freund tritt ein
- 5-Fragen-Quiz gegeneinander in Echtzeit
- Gewinner bekommt Münzen, Verlierer bekommt Trost-Bronze
- Rangliste der Freunde

### 11. Schreibschrift: Buchstaben als Beschwörung
Statt trockenem "Schreibe den Buchstaben nach":
- Jeder Buchstabe ist eine "Rune" die er meistert
- Die App zeichnet den korrekten Stift-Pfad vor (Animiert)
- Er folgt mit dem Finger/Stift nach
- Canvas erkennt wie nah er am Ideal ist → Sterne-Bewertung (1-3)
- Perfekte Rune = Spezialeffekt + Münzen

### 12. Trophäen-Zimmer / Helden-Galerie
Ein eigener Raum der automatisch befüllt wird:
- Erste englische Vokabel gelernt → "Sprachstein" Trophäe
- Ersten Mathe-Boss besiegt → "Zahlen-Schwert" Trophäe
- 7-Tage-Streak → "Eiserner Wille" Trophäe
- Sichtbar, dekorierbar, teilbar (Screenshot-Button)

## 🗄️ DATENBANK-SCHEMA (Supabase / PostgreSQL)

```sql
-- Benutzer
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  theme TEXT DEFAULT 'purple',       -- gewähltes Farbthema
  avatar_config JSONB DEFAULT '{}',  -- Outfit, Farbe, Accessoires
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Fortschritt pro Modul
CREATE TABLE module_progress (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  module TEXT NOT NULL,              -- 'english', 'math', 'hockey', etc.
  level INTEGER DEFAULT 1,
  xp INTEGER DEFAULT 0,
  lessons_completed INTEGER DEFAULT 0,
  last_played TIMESTAMPTZ,
  UNIQUE(user_id, module)
);

-- Münzen-Konto
CREATE TABLE coins (
  user_id UUID PRIMARY KEY REFERENCES users(id),
  bronze INTEGER DEFAULT 50,
  silver INTEGER DEFAULT 0,
  gold INTEGER DEFAULT 0
);

-- Gesamtrang & XP
CREATE TABLE hero_stats (
  user_id UUID PRIMARY KEY REFERENCES users(id),
  total_xp INTEGER DEFAULT 0,
  rank TEXT DEFAULT 'Schüler',
  rank_level INTEGER DEFAULT 1,
  current_combo INTEGER DEFAULT 0,
  best_combo INTEGER DEFAULT 0
);

-- Streak
CREATE TABLE streaks (
  user_id UUID PRIMARY KEY REFERENCES users(id),
  current_streak INTEGER DEFAULT 0,
  longest_streak INTEGER DEFAULT 0,
  last_login DATE,
  next_reward_at INTEGER DEFAULT 7    -- Tage bis nächstes Streak-Item
);

-- Inventar (gekaufte Items)
CREATE TABLE inventory (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  item_id TEXT NOT NULL,
  equipped BOOLEAN DEFAULT false,
  unlocked_at TIMESTAMPTZ DEFAULT now()
);

-- Shop-Items (wird durch seed.sql befüllt)
CREATE TABLE shop_items (
  id TEXT PRIMARY KEY,
  name TEXT NOT NULL,
  description TEXT,
  type TEXT NOT NULL,                -- 'outfit', 'accessory', 'room_deco', 'unlock'
  rarity TEXT DEFAULT 'common',     -- 'common', 'rare', 'epic', 'legendary'
  cost_type TEXT NOT NULL,          -- 'bronze', 'silver', 'gold'
  cost_amount INTEGER NOT NULL,
  preview_url TEXT,
  unlock_condition TEXT             -- NULL = sofort kaufbar
);

-- Missionen
CREATE TABLE missions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  mission_id TEXT NOT NULL,
  rank TEXT NOT NULL,               -- 'C', 'B', 'A', 'S'
  status TEXT DEFAULT 'active',     -- 'active', 'completed', 'claimed'
  assigned_at DATE DEFAULT CURRENT_DATE,
  completed_at TIMESTAMPTZ
);

-- Quiz-Verlauf
CREATE TABLE quiz_history (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  module TEXT NOT NULL,
  score INTEGER NOT NULL,
  max_score INTEGER NOT NULL,
  duration_seconds INTEGER,
  played_at TIMESTAMPTZ DEFAULT now()
);

-- Trophäen
CREATE TABLE trophies (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  trophy_id TEXT NOT NULL,
  unlocked_at TIMESTAMPTZ DEFAULT now(),
  UNIQUE(user_id, trophy_id)
);

-- Lernzeit-Tracking
CREATE TABLE session_tracking (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  module TEXT NOT NULL,
  started_at TIMESTAMPTZ,
  ended_at TIMESTAMPTZ,
  hour_of_day INTEGER,              -- 0-23 für Analyse
  score INTEGER
);

-- Boss-Kämpfe
CREATE TABLE boss_fights (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  week_number INTEGER NOT NULL,
  boss_name TEXT NOT NULL,
  damage_dealt INTEGER DEFAULT 0,
  won BOOLEAN DEFAULT false,
  completed_at TIMESTAMPTZ
);
```

## 🎮 MODUL-DETAILS

### Englisch (A1-A2)
**Lektions-Typen:**
1. Vokabel-Karten (Wort + Bild + Audio-Taste)
2. Lückentext-Übungen
3. Bild-Wort-Zuordnung
4. Einfache Satz-Übersetzung
5. "Tages-Geschichte" — kurze illustrierte Geschichte auf A1-Niveau

**Inhalt-Bereiche:**
- Grundvokabular (Zahlen, Farben, Familie, Schule)
- Begrüßungen und einfache Sätze
- Einfache Grammatik (to be, to have, Artikel)

### Mathe (Schulstufe-angepasst)
Der Junge wählt seine Klasse (5. / 6. Klasse).
**Themen pro Klasse:**
- Klasse 5: Grundrechenarten, Brüche, Dezimalzahlen, Geometrie-Basics
- Klasse 6: Prozente, Verhältnisse, negative Zahlen, Flächen/Volumen
**Aufgaben-Generator:** Algorithmus der zufällige aber passende Aufgaben erzeugt

### Hockey (Anfänger)
**Lektionen-Struktur:**
1. Ausrüstung kennenlernen (Schläger, Schutzausrüstung)
2. Grundregeln (Abseits, Ecke, Freischlag)
3. Basis-Techniken (Dribbling, Passtypen, Schuss)
4. Positionen auf dem Feld (Interaktives Taktik-Board)
5. Einfache Angriffsstrategien für Anfänger
6. Verteidigung: Grundprinzipien

### Auswendig-Lernen (Gedächtnis-Palast)
**Drei Räume:**
- Vokabeln → Hängen als "magische Schilder" an Wänden
- Formeln → Stehen auf leuchtenden Tafeln
- Texte → Liegen als aufgeschlagene Bücher auf Tischen
**Methode:** Spaced Repetition (Karteikarten-Algorithmus, SM-2)

### Schreibschrift (Canvas)
**Technik:**
- HTML5 Canvas + Pointer Events (Maus + Touch)
- Vorlagen-Pfad wird animiert vorgespielt
- Nutzer-Pfad wird aufgezeichnet
- Ähnlichkeits-Score berechnet (Punkt-zu-Punkt-Distanz)
- 1-3 Sterne Bewertung

### KI-Grundlagen
**Lernpfad:**
1. Was ist ein Computer? → Was ist ein Programm?
2. Was ist Daten? Wie lernt eine Maschine?
3. Wie funktioniert eine Bilderkennung? (visuelles Beispiel)
4. Was ist ein Sprachmodell? (vereinfacht)
5. Wie spricht man mit KI? (Prompting basics — sinnvoll!)
6. Wo ist KI in meinem Leben? (Handy, Spiele, YouTube-Empfehlungen)

**Didaktik:** Shin-Sensei erklärt mit simplen Analogien
(z.B. "Ein KI-Modell ist wie ein sehr fleißiger Schüler der Millionen Bücher gelesen hat")

### Quiz-Arena (Boss-Kampf-System)
**Modi:**
- Solo-Training: Gegen sich selbst (Highscore)
- Tages-Arena: Gemischte Fragen aller Module
- Boss-Kampf: Wöchentlich (Sonntag), Spezialgegner
**Frage-Formate:**
- Multiple Choice (4 Antworten)
- Wahr/Falsch
- Zeitrennen (30 Sekunden Countdown)
- Bild-Frage (Was ist das? Zeige Hockeytechnik)

## 🎨 DESIGN-SYSTEM

### Anime-RPG Ästhetik
- **Schriftart:** "Nunito" (rund, freundlich, lesbar) für Body
  + "Bangers" oder "Fredoka One" für Titel/Level-Anzeigen
- **Farbthemen (4 zur Auswahl):**
  - Lila (Standard): #7F77DD Haupt, #EEEDFE Hintergrund
  - Grün (Natur): #1D9E75 Haupt, #E1F5EE Hintergrund
  - Blau (Wasser): #378ADD Haupt, #E6F1FB Hintergrund
  - Rot (Feuer): #D85A30 Haupt, #FAECE7 Hintergrund

### Animations-Prinzipien
- Münzen fliegen mit Bogenbahn zum Münzzähler
- Level-Up: Lichtstrahlen + Konfetti + Rang-Text
- Combo: Zahl wächst mit Gummi-Effekt
- Falsche Antwort: Rotes Schütteln (Shake-Animation)
- Boss erscheint: Dramatischer Einzug von rechts

### Sound-Design
- Richtige Antwort: Kurzer Jubel-Ton
- Falsche Antwort: Tiefer Ton
- Münze erhalten: Klirr-Ton
- Level-Up: Fanfare
- Combo: Steigender Ton-Effekt
(Alle optional/mit Lautstärke-Regler)
